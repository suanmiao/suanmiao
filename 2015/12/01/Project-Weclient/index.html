Project-Weclient

2015-12-01 |

####Project brief

  • Description: WeChat official account management application (Opensource)
  • Platform: Android
  • Relating Techniques: Java, HTML DOM, Cookie Management, Regex
  • Duration: 2013.11 - 2015.5
  • Team member: Hongkun Leng(Developer), Hong Xiao(Product Designer), Bin Fan(Product Designer)

###Background:
Originally(before May 2015), there is no official application for account management, the only way was to login to official website, which was limited by location and time. However, most frequently behavior, message view and reply, happens randomly. So there is a high demand for account management features on mobile devices.

###Process

#####1.Developed login simulating framework

  • HTTP request management
  • MD5 encryption
  • Cookie management, parse Cookie, store Cookie, read Cookie

#####2.Developed Message magement function

  • Message list
  • message reply, star

#####3.Developed Fans management function

  • Fans list
  • Edit group, alias

#####4.Notification function

#####5.Opensource
Although it’s a fully functional application, it’s still unofficial, and we are only students whose credibility are limited. So, in order to show our honesty and enhance our credibility, we decide to publish our source code onto Github.

By now, without any promotion and advertisement, this opensource project has been starred by 130 developers and forked by 50 developers.
Really exciting, It’s the first time that I feel the super magic power of community, the influence of individuals can be amplified by the opensource community.
